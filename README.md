# Math Grid

A web application for math grid exercises.

## Running with Docker

Build the Docker image:
```bash
docker build -t sagrada-editor .
```

Run the container:
```bash
docker run -p 8080:80 sagrada-editor
```

Access the application at http://localhost:8080

## Development

The project structure:
- `source/` - Source files (HTML, CSS, JS)
- `dist/` - Built files (generated by build script)
- `scripts/build.sh` - Build script

To build manually:
```bash
bash scripts/build.sh
```

## Technologies

- HTML/CSS/JavaScript
- Docker (multi-stage build)
- Nginx (Alpine Linux)
